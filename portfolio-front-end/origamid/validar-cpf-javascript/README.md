Código com method's do JS que pegam CPF em qualquer formato do formulário e coloca no formato 000.000.000-00. Caso seja digitado com mais ou menos números, informa o usuário do erro, inclusive com leves alterações automáticas no CSS.

<hr>
<b>ENGLISH</b>

Code with JS method's that take CPF in any form format and place in the 000.000.000-00 format. If it is entered with more or less numbers, it informs the user about the error, even with slight automatic changes in CSS.
